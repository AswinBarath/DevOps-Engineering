1. Programming languages for DevOps

2. Server Administration
    - Linux (V.V.I)
    - Unix Systems
    - Windows server
3. Networking and Security
    - TCP/IP fundamentals
    - Protocols
4. Servers
    - Web Servers:
        - Apache
        - Nginx
        - Tomcat
    - Caching:
        - Redis

8. Cloud Computing
    - AWS
    - GCP
    - Azure
